# Felpudo - Final Project

## Project Overview

**Felpudo** is a 2D platformer game developed in **Unity** as a final project for the Game Programming course in the Digital Games graduation on UNEB - Universidade do Estado da Bahia. The game was developed by the students Jonas Barreto and Tha√≠s Ferreira.

The player controls Felpudo as he overcomes obstacles, avoids enemies, and manages limited resources to reunite with Fofura.

## Game Mechanics & Features

### Core Mechanics
* **Platforming:** Jump and walk controls to navigate the environment.
* **Resource Management (Level 2):** The player must collect **gasoline** to prevent the primary timer/fuel bar from hitting zero. On level 3 you must collect **fruits**.
* **Enemy Interaction:** Enemies deal damage on contact.

## Technical Details and Setup

### Technologies Used
* **Game Engine:** Unity (Version: 2022.3.52f1)
* **Language:** C#
